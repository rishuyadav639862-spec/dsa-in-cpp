# Shortcut keys

## File operations
- Ctrl + N = New file
- Ctrl + O = Open file
- Ctrl + S = Save file
- Ctrl + Shift + S = Save as
- Ctrl + W = Close file
- CTRL + Tab = Switch between files

---

## 2. Editing Code

- Ctrl + C = Copy
- Ctrl + V = Paste
- Ctrl + X = Cut
- Ctrl + Z = Undo
- Ctrl + Y = Redo
- Ctrl + / = Comment / Uncomment line

---

## 3. Code navigation

- Ctrl + P = Search file quickly
- Ctrl + F = Find in file
- Ctrl + H = Find & Replace
- Ctrl + G = Go to line number
- F12 = Go to defination
